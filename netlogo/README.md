# NetLogo ABM Model Instructions

## Installation

1. Download NetLogo 6.4 from https://ccl.northwestern.edu/netlogo/
2. Open `usle_abm_model.nlogo`

## Input Files

The model requires:
1. DEM file (.asc format) - generated by `generate_synthetic_dems.py`
2. Land-use file (.asc format) - generated by `generate_landuse.py`

## Running Simulations

1. Click "Setup" to load input files and initialize agents
2. Set rainfall depth (P) using the slider
3. Set soil type (Chernozems/Cambisols)
4. Click "Go" to run simulation
5. Results are exported to `output/` directory

## Parameters

See `../docs/parameters.md` for complete parameter descriptions.

## Output Files

- `erosion_map.asc` - Spatial distribution of erosion
- `summary_stats.csv` - Block-level statistics
- `flow_paths.png` - Visualization of flow routing
